route

Version 1.1

Route is a library for building and working with path base dispatch.

quick start

Getting started is pretty simple, just import the route package: 

import io.mth.route._

Build a route function: 

val route =
  ("book" <%> stringtoken).route(book =>
    Get(getbook(book)) |
    Put(createbook(book))
  ) | ("people"  "authors").route(
    Get(listauthors)
  )

Construct or parse a request object. 

val req = request(Get, "book"  "1984", text.html)

Apply request to route function. 

val res = route(req)
